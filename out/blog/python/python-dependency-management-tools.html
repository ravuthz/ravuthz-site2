Python Dependency Management with Peotry

Poetry helps us declare, manage and install dependencies of Python projects, ensuring you have the right stack everywhere. It's supports Python version 3.6+.

Installation

The complete documentation is available on the official website.

Linux / OSX / Bash on Windows ( WSL ) instructions

curl -sSL https://install.python-poetry.org | python3 -

Window - Powershell instructions

(Invoke-WebRequest -Uri https://install.python-poetry.org -UseBasicParsing).Content | python -

The installer install the peotry tool to bin directory.

  • On Unix at $HOME/.poetry/bin automatically added to your $PATH environment variable.
  • On Windiws at %USERPROFILE%\.poetry\bin automatically added to your %PATH% environment variable.

Usage

# Check installed version
poetry --version

# Add dependency
peotry install prettytable

The complete documentation is available on the official website.
